# module-minifier

A Node.js module minifier that uses Webpack and Terser to optimize your JavaScript packages.

## Installation

```sh
npm install module-minifier
```

## Usage

Example usage in your project:

```js
const minifier = require('module-minifier');
// Use the module APIs here
```

## API

See the TypeScript declaration files (`.d.ts`) in `lib/` for details on the exported APIs.

## Requirements

- Node.js >= 14
- npm >= 6
