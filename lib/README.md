# module-minifier

Un minificatore di moduli Node.js che utilizza Webpack e Terser per ottimizzare i tuoi pacchetti JavaScript.

## Installazione

```sh
npm install module-minifier
```

## Utilizzo

Esempio di utilizzo nel tuo progetto:

```js
const minifier = require('module-minifier');
// Usa le API del modulo qui
```

## API

Consulta i file di dichiarazione TypeScript (`.d.ts`) in `lib/` per i dettagli sulle API esportate.

## Requisiti

- Node.js >= 14
- npm >= 6
